# java-crud-jdbc

Projeto de estudo: CRUD em Java usando JDBC.

Há duas formas de executar:
- Modo sem banco (padrão): ideal para testar sem MySQL — basta rodar a classe Main.
- Modo com banco (JDBC): configure URL, USER e PASS no arquivo src/Main.java e crie a tabela `usuarios` no seu MySQL.
